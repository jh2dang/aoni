# 랭킹 및 캐릭터 비교 시스템

## 📊 랭킹 시스템

### 기능
- **서버별 랭킹 조회**: 서버 선택 드롭다운으로 각 서버의 랭킹 확인
- **시즌 정보 표시**: 현재 랭킹 시즌 정보 및 기간 표시
- **상세 정보**: 순위, 캐릭터명, 직업, 길드, 포인트, 순위 변동, 전투 기록(킬/어시스트/데스)

### 구현
- API: `/api/ranking/list` 엔드포인트 활용
- 상위 3위 특별 표시 (금/은/동 배지)
- 순위 변동 표시 (상승/하락 화살표)
- 캐릭터 클릭 시 상세 페이지로 이동

## 💾 캐릭터 저장 시스템

### 기능
- **내 캐릭터 저장**: 캐릭터 상세 페이지에서 "내 캐릭터로 추가" 버튼으로 저장
- **LocalStorage 활용**: 브라우저 로컬 스토리지에 캐릭터 정보 저장
- **저장 정보**: 캐릭터 ID, 서버 ID, 기본 정보, 장비 정보, 스킬 정보

### 구현
```typescript
// 저장
saveMyCharacter(characterId, serverId, infoData, equipmentData)

// 조회
getMyCharacter(): MyCharacter | null

// 확인
hasMyCharacter(): boolean
```

## ⚖️ 캐릭터 비교 시스템

### 기능
- **양방향 비교**: 비교 대상(왼쪽) vs 내 캐릭터(오른쪽) 카드 형식 비교
- **스탯 비교**: 모든 능력치를 개별 카드로 나누어 비교
- **장비 비교**: 슬롯별 장비 비교 (강화 레벨 차이 표시)
- **스킬 비교**: 보유 스킬 및 레벨 비교

### 차이값 표시
- **높음/낮음 표시**: 내 캐릭터가 비교 대상보다 높은지 낮은지 명확히 표시
- **차이값 계산**: `내 캐릭터 값 - 비교 대상 값` 기준
- **시각적 강조**: 차이가 큰 항목(10% 이상 또는 100 이상)은 배경색과 테두리로 강조
- **퍼센트 표시**: 차이가 큰 경우 퍼센트로 표시

### UI 특징
- 각 항목을 개별 카드로 분리하여 가독성 향상
- 왼쪽: 비교 대상 (녹색 테두리)
- 오른쪽: 내 캐릭터 (파란색 테두리)
- 차이값 배지: 높음(초록), 낮음(빨강), 동일(회색)

## 🔄 사용 흐름

1. **랭킹 페이지**에서 원하는 캐릭터 선택
2. **캐릭터 상세 페이지**에서 "내 캐릭터로 추가" 클릭
3. 다른 캐릭터 상세 페이지에서 "내 캐릭터와 비교" 버튼 클릭
4. **비교 페이지**에서 상세 비교 결과 확인

## 💡 기술적 특징

- **LocalStorage**: 클라이언트 사이드 저장소 활용
- **타입 안정성**: TypeScript로 저장/조회 데이터 타입 정의
- **반응형 디자인**: 모바일/데스크톱 모두 지원
- **시각적 피드백**: 차이값에 따른 색상 및 아이콘 표시
